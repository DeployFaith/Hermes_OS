# HermesOS

HermesOS is a Godot 4.6.3 project for an in-game operating system interface.

## Requirements

- Godot 4.6.3 or newer compatible 4.6 build

## Run

1. Open this repository in Godot.
2. Let Godot import the project assets.
3. Run the project. The main scene is `res://scenes/world_3d.tscn`.

## Project layout

- `project.godot` — Godot project configuration
- `scenes/` — game scenes
- `scripts/` — game and UI scripts
- `assets/` — game assets
- `addons/` — bundled Godot addons
- `content/` — bundled in-game content
